C++ 17 Relational Database Solution

Description:

A relational database management solution which utilizes a persistent 1KB block storage system and utilizes SQL commands to perform the following tasks:
  • Create, use, show, and drop databases.
  • Create, show, and drop tables.
  • Insert, update, and delete rows.
  • Query using select *, select "column" from "table", or select "table.column" to specify columns.
  • Filter queries with the keywords where, order by, group by, desc, limit, offset, and, or, and not.
  • Left joins.
